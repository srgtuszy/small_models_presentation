# Presentation Slides

## Viewing the Slides

### Option 1: Open Directly
Open `index.html` in any modern browser.

### Option 2: Local Server (Recommended)
```bash
# Python
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

Then open http://localhost:8000

## Navigation

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `↓` | Next sub-slide |
| `↑` | Previous sub-slide |
| `F` | Fullscreen |
| `S` | Speaker notes |
| `O` | Overview mode |
| `Esc` | Exit modes |

## Framework

Built with [Reveal.js](https://revealjs.com/) - the most popular open-source HTML presentation framework.

Features:
- Responsive design
- Keyboard navigation
- PDF export (print to PDF)
- Speaker notes
- Code highlighting
- Fragments (step-by-step reveals)
- Vertical slides for details
